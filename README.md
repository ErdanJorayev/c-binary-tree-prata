# Binary Search Tree Implementation (C)

This repository contains my implementation of a **Binary Search Tree (BST)**, inspired by the classic book *"C Primer Plus"* by **Stephen Prata**.

### About this project
After finishing Stephen Prata's guide to C, I found the BST implementation to be one of the most challenging yet fascinating parts of the book. 

* **The Logic:** This is a classic Binary Search Tree.
* **Modifications:** While I've kept the core logic and the "spirit" of the original code, I have made some personal adjustments to improve readability and structure.
* **My Impression:** I am genuinely impressed by how elegantly this structure handles data. It's a perfect example of how complex logic can be built using C's powerful features.

### Why this matters
Implementing this tree helped me deeply understand:
- Recursive algorithms in C.
- Advanced pointer manipulation.
- Dynamic memory management.
